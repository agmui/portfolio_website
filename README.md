# Portfolio Website

### link to website: https://portfolio-website-410a2.web.app/

![website preview](assets/2023-07-09_16-16.png)

requires Node.js version 18+. 20+
to use:  

`nvm use latest`

`npm install`  

`npm run dev`  

To deploy:

`firebase deploy`

> Note: you might need to do `firebase logout` and `firebase login`
